Your Projects

Etch-A-Sketch

This project is a browser-based Etch-A-Sketch built using HTML, CSS, and vanilla JavaScript. It dynamically generates a square drawing grid based on user input, allowing users to draw by simply moving their mouse over the grid. Each grid cell changes to a randomly generated RGB color on hover, creating a colourful sketching effect.

Calculator

This project is a basic on-screen calculator built with HTML, CSS (Flexbox), and vanilla JavaScript. It includes a button-based interface for digits (0-9), core operators (addition, subtraction, multiplication, division), an equals button to evaluate expressions, and an AC (clear) button to reset the calculator state.

Sign-up form

This project is a responsive sign-up page built with HTML and CSS. It features a split layout with a full-height background image and a modern registration form. Custom fonts are used to enhance visual identity, while CSS Flexbox ensures clean alignment and spacing throughout the layout. The form includes basic client-side validation using HTML attributes and CSS pseudo-classes, providing immediate visual feedback for valid and invalid inputs. The overall design focuses on clarity, accessibility, and a polished user experience.

Rock-paper-scissors

This project is a browser-based Rock–Paper–Scissors game built with HTML and JavaScript. The player chooses an option by clicking one of three buttons, while the computer makes a random choice each round. Results are displayed dynamically on the page using DOM manipulation, including both the player’s selection and the computer’s selection. The game tracks scores across rounds with variables for the human and computer, updating and showing the current score after every play.

Odin Landing Page

This project is a simple landing page built with HTML and CSS, styled using the Roboto font. It includes a full hero/header section with navigation links, a call-to-action button, and an image. The layout is structured using Flexbox to align elements cleanly across sections. The page also features an information grid with image cards, a testimonial/quote section, a highlighted blue call-to-action banner, and a footer. Overall, it focuses on practicing modern layout, spacing, typography, and reusable styling.

Horse Race Terminal

This project is a console-based horse race simulator written in Java. It uses object-oriented programming by modelling each horse as an object with its own name, symbol, confidence level, distance travelled, and “fallen” state. The race runs in timed rounds where each horse either moves forward based on a random step influenced by its confidence or has a small chance of falling. The program continuously prints the race track to the console, updates positions in real time, and ends by declaring a winner when a horse reaches the finish line (or no winner if all horses fall).
